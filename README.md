# Revenue Agent Lite

Free Claude skill for a basic daily revenue briefing and business overview.

## Features
- Generate a daily briefing of your online business status
- Track key metrics across properties
- Quick snapshot of revenue and traffic trends

## Installation
Download the `.skill` file and install it in Claude Desktop (Cowork mode).
